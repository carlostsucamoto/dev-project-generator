# Dev Project Generator

Gerador de ideias de projetos para desenvolvedores. Sorteia projetos, desafios e sugestões de estudo com base em linguagens e bibliotecas.

## Tecnologias
- Python
- SQLite
- Dash

## Como executar
1. Clone o repositório
2. Execute `python seed.py` para popular o banco
3. Execute `python app.py` para iniciar a aplicação
